# ShadowsocksFree

之所以有做各样一个应用的想法是因为搬瓦工前几天电信出口炸了半天，当时萌生了一个App端获取一些免费账号过渡搬瓦工不稳定时期的想法，正好想学Swift，于是做了下，把想学的都加进去玩。
